About CryptoPred

This application provides cryptocurrency price predictions using a sophisticated backend stack designed for performance and scalability. We utilize Long Short-Term Memory (LSTM) models, a type of recurrent neural network (RNN), to analyze historical price data and forecast future trends.

Technology Stack

  • Backend: Flask, a lightweight and flexible Python web framework.
  • Asynchronous Tasks: Celery, for running the prediction models in the background to prevent UI blocking.
  • Data & Caching: Redis, used as the high-performance message broker for Celery and for session management.
  • Frontend: Modern HTML, CSS (Tailwind), and vanilla JavaScript for a dynamic single-page application experience.
  • Data Source: Real-time market data is sourced from the official Binance API.
